# ECHO FILM Memorial Archive

## 한국어

이 저장소는 조은령 감독 추모 홈페이지의 공개 보존본입니다.

이 아카이브는 [www.echofilm.com](https://www.echofilm.com) 내용을 가능한 오래 접근할 수 있도록 남겨 두기 위한 백업 저장소입니다.

같은 보존 목적의 공개 주소로 [echofilm.pages.dev](https://echofilm.pages.dev) 도 함께 운영되고 있습니다.

## English

This repository is a public preservation copy of the memorial homepage for director Eun-ryung Cho.

It exists as a backup of [www.echofilm.com](https://www.echofilm.com), so the site and its contents can remain accessible for as long as possible.

Another public archive for the same preservation purpose is also available at [echofilm.pages.dev](https://echofilm.pages.dev).
